How this website is made

project1 is a fullstack game review platform: backend serves pages and APIs, PostgreSQL stores users/reviews/thread data, and SerpAPI enriches searches with YouTube review media.

Architecture

  • Frontend: vanilla HTML, CSS, and JavaScript.
  • Backend: Node.js + Express with cookie-based session auth.
  • Database: PostgreSQL schema in data/supabase.sql and data/supabase-reviews.sql.
  • Crawler: SerpAPI YouTube engine, with stored metadata for diagnostics.

Security and data handling

  • Passwords are hashed in Postgres with pgcrypto crypt() and never returned by APIs.
  • Session tokens are random, stored as SHA-256 hashes, and set in HttpOnly cookies.
  • Input is normalized and length-bounded before writes.
  • Review authors are masked in response payloads.

Feature map

  1. Search game in Home or Dashboard.
  2. Read community reviews and star ratings.
  3. Watch fetched YouTube review videos.
  4. Post a review and reply inside game thread comments.
