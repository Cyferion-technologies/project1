About • MVP • Roadmap

project-1

We’re building a simple game review experience: search a game, view community reviews, pull YouTube review videos related to that game, and let users write and publish their own review.

MVP (what we’re building first)

  • Search a game and resolve it to a single result.
  • Show community reviews for that game.
  • Pull YouTube review videos for the searched game and show links.
  • Allow a signed-in user to write and post a review for that game.
Team
  • Benjamin Oliveira
  • Scott Palma
  • Darius Pal

Current roadmap

  1. Wire up the game search so users can reliably find the intended title.
  2. Show stored reviews and enable posting new reviews (auth + validation).
  3. AI crawler: query YouTube review videos for the searched game and return top matches.
  4. Display those videos as clickable links/cards near the game results.
  5. Connect the flow: after watching a video, the user can write a review for the same game.
